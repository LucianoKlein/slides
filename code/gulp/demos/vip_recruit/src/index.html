疑问解答

1. 课程周期是多久?
课程学习周期是4-5个月
2. 怎样上课?
会议软件超清在线直播：工作日上2个晚上，每次2小时，周末上1个晚上，每次3小时；每周5~7课时直播。 直播录屏复习，并完成课后作业，提交并批改。 教学管理：班主任会监督管理学员学习状况，包括直播课是否参加，作业是否过关等。 技术辅导：学习期间，任何技术问题都可以询问讲师，包括班级QQ群、私聊。语音等。毕业之后进入校友群 享受永久优先答疑服务。
3. 零基础可以学习吗?
可以的，不过平时得多花点时间。
4. 已经工作了，可以选课学习吗?
可以的，选课方式找教导员咨询QQ: 1668537174。
5. 有项目吗？有就业指导吗？
课后练习有小项目，课程尾声会安排大家做毕业设计，由老师指导，简历面试指导通通都有嗒~。
6. 不想学了可以退课吗?
支持随心退。学习的任何阶段都支持无条件退课退费，所退费用=总费用-已上课时费用
7. 手头有点紧，支持分期吗?
支持分期支付课时费，详询教导员~

每周三、五、日20:30-22:30点上直播课，课下需要完成当次课安排的任务，每天晚上有自习，任何时间有问题都可以得到老师的辅导
