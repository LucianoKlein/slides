CSS 是什么

CSS全称是 Cascading Style Sheets, 层叠样式表

@charset "utf-8";
h1 {
  color: red;
  font-size: 20px;
  /*这是注释*/
}
a:hover{
  color: red;
}

如何在页面引用 CSS

内联样式

<h1 style="color: red; font-size: 20px;"></h1>

内部样式


<style type="text/css">
  h1 {
    color: red;
    font-size: 20px;
  }
</style>
<h1>饥人谷</h1>

外部样式

<head>
  <link rel="stylesheet" type="text/css" href="index.css">
</head>

<style>
  @import url("hello.css");
  @import "world.css";
</style>

书写规范

  • 语法不区分大小写，但建议统一使用小写
  • 不使用内联的style属性定义样式
  • id和class使用有意义的单词，分隔符建议使用-
  • 有可能就是用缩写
  • 属性值是0的省略单位
  • 块内容缩进
  • 属性名冒号后面添加一个空格
